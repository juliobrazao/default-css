Nesting as the main-key

A short CSS library to help a HTML writing without a lot of classes and ids.




Too lightweight to be modular

Using a simple nesting structure, you only have to set the corrects sections to have a ordened page.

Customizable where it needs

This short CSS code, gives to you the possibility of change anything without get lost in a bunch of classes and ids.

Easy as fry an egg!

Created to reduce even more the process of CSS diagramation, preventing we all to cross a project final deadline due the worst part of development.
